①Linux: あるディレクトリから再帰的にファイル一覧を取得するコマンドを実行してください。必要に応じて調査しつつ実施お願いします。
→ls -R

②Windows: 同様にWindowsコマンドであるディレクトリから再帰的にファイル一覧を取得するコマンドを調査してください。(実務ではLinux, Windowsそれどれで同様の処理をおこなうことがあります。)
→dir /s

③Linux: ファイルが格納されているディレクトリを一気に削除するコマンドを実行してください。
→rm -r

④(応用)Shellスクリプトについて調査して概要を報告してください。Shellスクリプトで任意のLinuxコマンドを実装し実行してください。
→関数や演算などの処理を記述しておいて実行できるファイル


⑤Linux: ファイルの書き込み権限等含めてリストアップするコマンドを実行してください。
→ls -l


⑥(応用) emacsについてインストール可能ならばインストールし、実際にemacsでテキストを追加・保存してください。


⑦Linux: chmodの755について説明してください。実際にコマンドを実行して元の状態からどう変化するか確認してください。
→chmod:フォルダ、ファイルの権限を変更する。１桁目：所有者の権限　２桁目：所有者グループの権限　３桁目：それ以外の権限　
各桁の数字は、読取権限：４　書込み権限：２　実行権限：１で表現される。複数権限は和で表現する。
chmod755は対象ファイルの権限を　所有者：全て　所有者グループ：読取、実行　その他：読取、実行に変更する。

⑧git: commitとpushの違いについて説明してください。
→commit:インデックスに登録された変更箇所をローカルリポジトリに記録する。　
　push:ローカルリポジトリの変更内容をリモートリポジトリに反映する。

⑧github&git: (操作の慣れ)github上・gitコマンド経由それぞれでファイルの削除・追加・変更を複数回実行してください。(基本的に、最も利用するのは追加・変更・削除ですので、この操作を自然とできるようにしましょう)

⑨(応用)Linux: chownについて調べ実行した結果を報告してください。どのようなコマンドか説明してください。
→ファイルまたはディレクトリの所有権を変更するコマンド
